INSTRUCOES 
1. Baixe a skill do repositório
Acesse o repositório de skills disponibilizado pela sua equipe e faça o download da pasta da skill desejada. Cada skill é uma pasta contendo um arquivo SKILL.md.

3. Faça o upload da skill no Claude
Na interface do Claude, clique no ícone de ferramentas ou configurações e localize a opção de upload de arquivos de skill. Selecione a pasta ou o arquivo SKILL.md que você baixou e confirme o envio.

5. Verifique se a skill foi carregada
Após o upload, a skill deve aparecer na lista de skills ativas. Confirme que o nome bate com o que você baixou do repositório.

7. Teste a skill
Inicie uma nova conversa e faça um pedido que deveria acionar a skill. O Claude deve responder seguindo as instruções da skill.



O QUE CADA SKILL FAZ?

ESTRATEGISTA
Estrutura problemas de design de produto antes que as soluções existam. Sintetiza pesquisas, dimensiona oportunidades, define hipóteses, delimita o escopo de projetos e mapeia jornadas do cliente.   **Use esta habilidade para:** *Kickoffs* de novos projetos, solicitações de negócios ambíguas, tradução de pesquisas em *briefings*, sessões de enquadramento estratégico, avaliações de oportunidade, definição de escopo, alinhamento de *stakeholders* e análise competitiva — mesmo que o usuário não diga explicitamente a palavra "estrategista".

ANALISE COMPETITIVA
Realiza analise competitiva de produtos digitais de edtech e saude, comparando plataformas a partir de prints de telas e descricoes textuais. Use esta skill SEMPRE que o usuario pedir para analisar, comparar ou avaliar plataformas de ensino medico, cursos preparatorios, bancos de questoes, LMS de saude, ou qualquer produto digital voltado para educacao e saude. Tambem acione quando o usuario mencionar: analise de concorrente, benchmarking de produto, comparativo de plataforma, mapeamento de features, gaps estrategicos, oportunidades de produto, ou variacoes como analisa essas telas, compara essas plataformas, o que o concorrente tem que a gente nao tem.

FLUXO DE DESIGNER 
Projete qualquer experiência voltada ao usuário de ponta a ponta: fluxos de tarefas, fluxos de trabalho de várias etapas, estruturas de navegação, onboarding, configurações, busca, criação de conteúdo, colaboração, cadastro, checkout, dashboards, notificações, recuperação de erros e muito mais. Gerencia a adaptação multiplataforma (mobile/web/TV/embarcados), design responsivo ao dispositivo, acessibilidade, especificações de interação e sistemas de tipografia.   Acione este agente ao projetar fluxos de usuário de qualquer tipo, mapear sequências de telas, otimizar a conclusão de tarefas, especificar interações, projetar navegação ou perguntar "como deve ser a experiência do usuário em X?". Também deve ser acionado para sistemas de tipografia, seleção de fontes, pareamento de fontes, hierarquia tipográfica ou "quais fontes devemos usar?". Use esta habilidade de forma abrangente — sempre que alguém estiver trabalhando em como um usuário navega por uma experiência de produto, esta habilidade se aplica.

ARQUITETO DE SISTEMAS
Mapeia, analisa e projeta os sistemas por trás das experiências de produto. Cria *service blueprints*, mapas de ecossistema, arquitetura de processos e diagramas de dependência. Compreende como serviços, equipes, ferramentas e fluxos de dados se conectam para gerar (ou falhar em gerar) resultados para o usuário. Propõe mudanças estruturais na forma como produtos e serviços são organizados.   **Acione este agente para:** *Service blueprints*, mapas de sistema, arquitetura de processos, mapeamento de atores/papéis, análise de dependência, fluxos de trabalho multifuncionais, design operacional ou perguntas como "como este sistema funciona?", "o que quebra quando X acontece?", "mapeie o serviço", "onde estão as dependências?" ou qualquer questão sobre a engrenagem estrutural por trás de uma experiência de produto. Use esta habilidade de forma abrangente — sempre que alguém precisar entender ou redesenhar como um sistema funciona, e não apenas o que o usuário vê.

TESTE DE USABILIDADE
Realiza testes de usabilidade profissionais com personas sintéticas para análise de interfaces digitais (websites, apps, dashboards, mobile).  Use esta skill SEMPRE que o usuário pedir para testar, avaliar, analisar ou auditar uma interface de usuário,  produto digital ou fluxo de UX. Também acione quando o usuário mencionar: walkthrough cognitivo, avaliação heurística,  teste de acessibilidade, resposta emocional, personas de usuário, UX research, usabilidade,  fricções de interface, onboarding, conversão, jornada do usuário, Nielsen, WCAG, NPS de produto,  ou qualquer variação de "como meu usuário usaria isso". Use mesmo que o usuário não mencione explicitamente "skill" ou "teste de usabilidade" —  se o contexto envolver avaliar uma interface pela ótica de um perfil de usuário, esta skill se aplica.


HANDOFF
Enviando links do figma ou Prints de protótipos faz a ponte entre o design e a engenharia, produzindo especificações detalhadas, pacotes de entrega organizados, inventários de ativos (assets) e documentação multifuncional.   **Acione este agente ao:** Escrever especificações de design, preparar pacotes de entrega para engenharia (handoff), documentar para o desenvolvimento, criar revisões de design (design reviews), escrever planos de testes A/B, construir matrizes de texto (copy matrices), tratar casos de borda (edge cases), alinhar stakeholders, empacotar designs "para a engenharia" ou dizer "escreva a especificação", "prepare o handoff", "documente isso" ou "do que precisamos para a revisão de design?".
