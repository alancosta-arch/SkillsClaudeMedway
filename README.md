Como adicionar uma skill no Claude
Skills são instruções personalizadas que ensinam o Claude a se comportar de forma específica em determinados contextos. Siga os passos abaixo para instalar uma skill.

1. Baixe a skill do repositório
Acesse o repositório de skills disponibilizado pela sua equipe e faça o download da pasta da skill desejada. Cada skill é uma pasta contendo um arquivo SKILL.md.

3. Faça o upload da skill no Claude
Na interface do Claude, clique no ícone de ferramentas ou configurações e localize a opção de upload de arquivos de skill. Selecione a pasta ou o arquivo SKILL.md que você baixou e confirme o envio.

5. Verifique se a skill foi carregada
Após o upload, a skill deve aparecer na lista de skills ativas. Confirme que o nome bate com o que você baixou do repositório.

7. Teste a skill
Inicie uma nova conversa e faça um pedido que deveria acionar a skill — por exemplo, se a skill for de relatórios financeiros, escreva algo como "crie um relatório financeiro mensal". O Claude deve responder seguindo as instruções da skill.
